# Full Intervention Results

This page provides the complete quantitative results for **Acceptance**, **Lift**, and **Adoption** across all model families. To keep the tables readable on GitHub, the original paper table has been reformatted into one table per model.

**Notation**
- **Sup@R1**: supportive knowledge targeting the model's original top-ranked choice
- **Op@R2–R5**: opposing knowledge targeting lower-ranked alternatives
- **Base, Acceptance, Lift** are reported as **Median [Q1–Q3]**
- **Adoption** is reported as **Mean ± SD**
- Confidence bins use the paper definitions: **low** (`κ < 0.65`), **mid** (`0.65 ≤ κ ≤ 0.85`), **high** (`κ > 0.85`)
